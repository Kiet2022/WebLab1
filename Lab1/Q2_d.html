Images and Links in HTML

HTML Links

Hyperlinks Syntax: The HTML < a > tag defines a hyperlink. It has the following syntax:

< a href="url">link text< /a>

Example: Visit w3schools.com!

Images in HTML

  1. Set background image for the page

    2. < body background="filename" > < /body>

  2. Insert images into Folder

    3. < img src=""link to folder" title="Note">

  3. Insert images into Web pages

    4. < img src="Coppy image address" title="Note">
